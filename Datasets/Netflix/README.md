# Netflix dataset

This dataset consists of tv shows and movies available on Netflix as of 2019. The dataset is collected from Flixable which is a third-party Netflix search engine.
In 2018, they released an interesting report (https://flixable.com/netflix-museum/) which shows that the number of TV shows on Netflix has nearly tripled since 2010.
